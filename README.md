# Derp Flipbook v3 - True Flipbook

A clean, mobile-first, frame-by-frame rebuild.

## Features
- Projects saved locally
- IndexedDB frame storage instead of localStorage
- Custom canvas size up to 3000x3000
- Up to 10,000 frames
- FPS up to 30
- Add frames with +
- Duplicate/delete frames
- Brush, eraser, fill, pan tools
- Color wheel / color picker + swatches
- Onion skin
- Two-finger canvas pan
- Sound attachment and preview
- Project JSON backup/import
- Video export using browser MediaRecorder
